# Product Image Slots

Upload the product images with these filenames, then the landing page will use them automatically after the kitchen brush content is applied from Admin -> Product.

- `kitchen-brush-hero-drain.jpg` - use the last image, brush dripping water. This appears in the hero/top area.
- `kitchen-brush-frypan-foam.jpg` - brush cleaning the black pan with foam.
- `kitchen-brush-pan-cleaning.jpg` - brush inside the pan.
- `kitchen-brush-hanging-storage.jpg` - hanging/storage image.
- `kitchen-brush-plate-demo.jpg` - plate cleaning image, used as the circular hero image and usage image.
- `kitchen-brush-pan-close.jpg` - close-up pan/brush image.

You can also skip these filenames and paste full image URLs in Admin -> Product.
